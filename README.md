# vue_onlineshop_adminsystem

## Project setup

```
npm install
```

### Compiles and hot-reloads for development

```
npm run serve
```

### Compiles and minifies for production

```
npm run build
```

### Lints and fixes files

```
npm run lint
```

## 数据库连接

1. 下载 phpstudy，启动 MySQL
2. 在 vue_api_server 文件夹下，打开 terminal，输入 `node .\app.js`
